shadowsocks

2 Sep 2015 in technology

"Two days ago the police came to me and wanted me to stop working on this" - The code of shadowsocks, a secure socks5 proxy, has been removed due to pressure from the Chinese government. The GitHub repository is now empty. Good that GitHub still keeps the history :)

There's also a discussion on Hacker News and a backup of all repos.
